## Useful Keys
 - `C-h` (`m` - describe modes, `f` - describe function, `v` - describe
variable, `k` - describe key, `a` - search by phrase, `w` - shows the key shortcut for the command (alternative for `M-x where-is`), C-a` - show home page)
